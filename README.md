# Sales Data Analysis — Superstore Dataset

## Overview
Exploratory data analysis on a real US retail superstore dataset with 9,994 sales transactions across 4 regions and 3 product categories.

## Tools Used
- Python 3.14
- Pandas — data loading and manipulation
- Matplotlib & Seaborn — data visualization
- Jupyter Notebook

## Dataset
Superstore Sales Dataset from Kaggle — 9,994 rows, 21 columns covering orders, customers, products, regions and sales figures from 2016–2019.

## Key Findings
- Canon imageCLASS 2200 Advanced Copier was the highest selling single product at $60,000+
- West region led all regions with $700,000+ in total sales, followed closely by East
- South region was the weakest market with ~$390,000 in sales
- Technology category contributed the highest sales share at 36.4%, followed by Furniture at 32.3%

## Charts Generated
- Top 10 products by total sales (horizontal bar chart)
- Total sales by region (bar chart)
- Sales share by product category (pie chart)

## How to Run
1. Clone this repo or download the files
2. Install dependencies: `pip install pandas matplotlib seaborn jupyter`
3. Open `sales-analysis.ipynb` in Jupyter Notebook
4. Run all cells in order with Shift + Enter
